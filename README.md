# Videx – Video Platform

A modern video discovery platform with a clean frontend/backend separation.

## Project Structure

```
videoplatform/
  frontend/    ← Next.js 14 app (UI, components, pages)
  backend/     ← Express.js API server (API routes, data, lib utils)
  videx/       ← (old folder — can be deleted once frontend is confirmed working)
```

## Running the Project

### Frontend (Next.js)
```bash
cd frontend
npm install
npm run dev
# → http://localhost:3000
```

### Backend (Express)
```bash
cd backend
npm install
npm run dev
# → http://localhost:4000
```
